### 닫은 a11y 부채 (B.4 leftover)

2. **Global `@media (prefers-reduced-motion: reduce)` 블록**을 `globals.css` EOF에 신설.
   - 대상: `.product-shot-dot`, `.statusbar .live .pulse`, `.product-chrome .right .pulse`, `.ticker-inner`, `.ticker-track`, `.lb-v3-foot .left .pulse`, `.activity-feed .af-head .pulse`
   - 처리: `animation: none !important`
   - 기존 component-local 블록(`.upload-success-card`, `.upload-success-banner`)은 **건드리지 않음** — 향후 component-specific rule 추가 여지 보존.

### Codex REVISE에서 잡은 함정

3. **Ticker offscreen-start trap** (TIL: `docs/til/2026-05-26-ticker-offscreen-start-trap.md`)
   - `.ticker-inner` / `.ticker-track`는 `padding-left: 100%` + `animation: ticker` 으로 content를 viewport 바깥에서 시작시킨 뒤 keyframe이 스크롤-인 시키는 marquee pattern.
   - `animation: none !important`만 적용하면 strip이 **영구 공백** → 기존 a11y 부채보다 더 나쁨.
   - 수정: 같은 reduced-motion 블록에서 `padding-left: 0 !important; transform: none !important;` 함께 적용해 static-layout fallback 유지.
   - Regression guard: `e2e/hero-pulse.spec.ts` test #4 (computed paddingLeft/transform + first-child viewport intersection 검증).

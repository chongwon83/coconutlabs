## 가정 및 제약

- **Estimated 등급**: Claude Code CLI·Codex CLI 모두 로그에 네이티브 `cost`/`usd`
  필드가 없어, 비용은 토큰 수 × `model-pricing.json` 단가로 산출한 추정치.
- **단가 변동**: 단가 기준일 2026-05-18. 공식 단가 변경 시 재계산 필요.
- **스킵 73파일**: 빈 파일 또는 `token_count` 이벤트 부재로 파싱 불가 — 누락이
  아니라 토큰이 0이거나 측정 불가한 세션.
- **세션 1파일 = 1세션** 가정으로 파일별 합산 → 이중 집계 없음.
- **ChatGPT 웹/앱 사용량 미포함**: 로컬 로그를 남기지 않아 PoC 범위 밖.
  `gpt-*` 행은 Codex CLI의 `model` 필드 기준.

## 산정 방법

- Claude: `type:"assistant"` 라인의 `message.usage` per-message 합산.
  `<synthetic>` 모델은 실제 모델명을 덮어쓰지 않음 (오분류 방지).
- Codex: 세션 최종 `token_count` 이벤트의 `info.total_token_usage` (누적값).
  `input_tokens`에서 `cached_input_tokens`를 분리해 이중 과금 방지.
- 재현: `./estimate_cost.py --all` (또는 `--all --json`).
